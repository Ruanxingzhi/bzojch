BZOJCH

BZOJCH是BZOJ题库open_in_new的离线版。

项目托管在github：ruanxingzhi/bzojchopen_in_new

什么是BZOJCH

如果bzoj挂掉了、用户没网、或者您嫌BZOJ难看，那么BZOJCH可以派上用场。

BZOJCH爬取了bzoj的题面，保存在本地备用。

反馈与bug提交

请联系：ruanxingzhi@gmail.com

BZOJCH用到的开源项目

  • Daemonite's Material UI
  • docker
  • Apache + PHP + MYSQL
